# FakeNewsSimulator

## Requirements

- Python 3.8
- Following packages
  - spade
  - numpy
  - networkx
  - matplotlib
  - flask
  - dash
  - plotly
  - sklearn
  - pandas

## How to run

First, start the visualization server:

```
python visualization/server.py -h
```

Then, start the simulation:

```
python main.py -h
```
